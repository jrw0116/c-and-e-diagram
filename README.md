# Cause & Effect Diagram

Cause & Effect Diagram is a lightweight fillable web app based on a six-section fishbone template.

## Included sections

- Equipment
- Process
- People
- Materials
- Environment
- Management

## What it does

- Captures the effect or problem being analyzed
- Provides fillable notes plus repeatable cause and sub-cause rows for each section
- Updates a fishbone-style diagram preview live as the form changes
- Saves progress locally in the browser
- Imports and exports the diagram as JSON
- Supports printing the diagram view

## Run it

Open [index.html](./index.html) in a browser.
